# 三林软件园班车时刻表

通过telegram bot获取实时的班车时刻表：

```bash
/get 获取全部路线
/bus/:lineNo 获取`${lineNo}`号线时刻
```

机器人名称：[bus_sanlin_bot](t.me/bus_sanlin_bot)

For a description of the Bot API, see this page: <https://core.telegram.org/bots/api>
